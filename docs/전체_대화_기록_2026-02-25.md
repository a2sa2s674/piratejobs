### 9. 성장 프레임워크 분석

**(1) MAU 분석 프레임워크**: 공급(Supply) → 수요(Demand) → 전환(Conversion)

**(2) 등급 시스템 데이터** (L1~L5):

- L2(Active): 900명 (14.5%)

- L4(VIP): 1,218명 (19.6%)
- L5(Soul): 692명 (11.1%)

**(3) 핵심 발견**:

- 강등률 44.3% (2,758명) — 포인트 전환 기준이 구조적 강등 원인
- 승급률 55.7% (3,464명)

**(4) KPI 목표**:
- L1→L3 성장
- 포인트 전환 경험률 74% → 90%
- MAU 32,149 → 38,000
- 강등률 44.3% → 35%
- L3+ 55.4% → 65%

**(5) 퀵스타트 온보딩**:
- D+0: 첫 설문 참여
- D+7: 포인트 전환 경험
- D+14: 2회 이상 참여
- D+30: 정기 참여 패턴 형성

**(6) 월간 챌린지**: 서베이 마라톤, 개근왕, 댓글 히어로, 공유 미션

**(7) 수익 시뮬레이터 UI 컨셉**

**8명의 에이전트 검증 결과**: 데이터 기반 접근이 강점. 3가지 보강 제안:

### 10. PMI 용어 정정

**박현우**: "PMI 서베이: QO, SO < CO를 늘려야함 — qo: 쿼터아웃, so: 스크리닝 아웃 = 포인트를 적게 받는 경험을 함. co : 컴플리트 아웃 = 포인트를 많이 받는 긍정적인 상황. 이 부분 정정해줘"

| 용어 | 의미 | 패널 경험 |
|------|------|-----------|
| QO (Quota Out) | 할당량 초과로 탈락 | 포인트 적게 받음 → 부정적 |
| SO (Screening Out) | 적격 심사 탈락 | 포인트 적게 받음 → 부정적 |
| CO (Complete Out) | 설문 완료 | 포인트 많이 받음 → 긍정적 |

**핵심 인사이트**: QO/SO가 많고 CO가 적으면 패널이 "헛수고" 경험만 반복 → 이탈. CO율 향상이 리텐션·MAU·매출의 공통 해법.

**8명의 에이전트 CO율 중심 재분석**:
- 전략 총괄: CO율 향상 = 리텐션 = MAU = 매출의 직접 연결고리
- 시장 분석가: 업계 CO율 30~40%, QO+SO 60% 이상이면 만족 유저 11,200명 이하
- 재무 관리자: CO율 높은 패널풀은 프리미엄 단가 가능
- 운영 관리자: 공급 측(서베이 품질) + 수요 측(패널 매칭) 양면 개선
- 마케팅 책임자: 첫 설문 = 반드시 CO 보장 필수
- 영업 책임자: CO율 70%+ 보장 = B2B 영업 무기
- 인사 책임자: 이소연(데이터 대시보드) + 김지우(품질 가이드라인) 역할 분담
- 기술 책임자: 프로필-서베이 매칭 알고리즘, 스크리닝 예측 모델, 실시간 대시보드 제안
