## 1. 프로젝트 개요

- **프로젝트명**: Agent Dream Team (사업 능숙팀)
- **스택**: Next.js 16 + TypeScript + Tailwind CSS
- **8명의 에이전트**: 전략 총괄, 시장 분석가, 재무 관리자, 운영 관리자, 마케팅 책임자, 영업 책임자, 인사 책임자, 기술 책임자

## 4. 핵심 논의 내용

### 4-1. 현황 진단

- **틸리언프로**: 10년+ 미개편, 경쟁사 대비 열위, 연평균 사외매출 8,300만원
- **헤이폴**: 2025.12.24 리팩토링 런칭 완료, 최신 인프라(K8s, Redis, AWS), AI(HyperCLOVA X)
- **패널**: 38,000명 활성, DAU 28,000~29,000명 (DAU비율 74~76%)

### 4-2. 전략 방향

- 틸리언프로 적극 영업 포기, 기존 매출 유지만
- 헤이폴을 B2C → B2B+B2C 데이터 수집 플랫폼으로 전환
- 경쟁사 픽플리(pickply.com) 벤치마킹
- **이중 트랙 전략**: Track A(즉시 매출 1.18억) + Track B(구조적 성장 0.92억) = 2.10억

### 4-3. Track B 신규 기능 (순차 개발)

1. 외부 폼 연동 서비스 (5월)
2. 셀프 프로젝트 등록 (6월)
3. 참여자 모집 IDI/FGI (7월)
4. CPA 광고 상품 (8월)
5. AI 보고서 구독형 (10월)

### 4-4. 성장 프레임워크 분석

- L1 유저 86.6%가 포인트 전환 경험 없음
- 강등률 44.3% — 포인트 전환 기준이 구조적 원인
- PMI 용어 정정: QO(Quota Out), SO(Screening Out) = 저보상, CO(Complete Out) = 고보상
- CO율 향상이 리텐션·MAU·매출의 공통 해법

### 4-5. 선순환 구조

- 학술 연구자(외부 폼) → 마케터(셀프서베이) → UX 리서처(IDI/FGI) → 기업(구독형)
- 오늘의 학생 유저 = 내일의 기업 고객 (직군 확장 선순환)
